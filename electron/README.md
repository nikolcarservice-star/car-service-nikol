# Nikol CRM — как собрать программу (.exe) на Windows

## Шаг 0. Установить Node.js (один раз)

1. Откройте в браузере: **https://nodejs.org**
2. Скачайте версию **LTS** (зелёная кнопка).
3. Запустите установщик и нажимайте «Next», пока установка не закончится.
4. Перезапустите компьютер (или закройте и снова откройте окно команд/терминал).

---

## Способ 1: Сборка двойным щелчком (проще всего)

1. Откройте папку проекта **Car Service Nikol**.
2. Зайдите в папку **electron**.
3. Дважды щёлкните по файлу **Собрать программу.bat**.
4. Дождитесь окончания (в чёрном окне пойдёт загрузка и сборка, может занять 2–5 минут).
5. Когда появится «Готово!», откроется папка с готовыми файлами **на диске D:** в каталоге **D:\nikol-crm-build\dist**:
   - **Nikol CRM Setup 1.0.0.exe** — установщик (запустите его, чтобы установить программу на компьютер);
   - **Nikol CRM 1.0.0.exe** — портативная версия (работает без установки).

**Важно:** кэш npm и Electron, а также готовые .exe сохраняются **на диск D:** (папка `D:\nikol-crm-build`), чтобы не занимать место на C:. Если нужно использовать другой диск (например E:), откройте **Собрать программу.bat** в блокноте и в начале измените строку `set "BUILD_DISK=D:"` на `set "BUILD_DISK=E:"`.

### Если появляется ошибка "Cannot create symbolic link"

Это значит, что Windows запрещает `mklink` (символические ссылки), а electron-builder использует их при сборке.

Сделайте одно из двух:

1. **Включите Developer Mode**:  
   Параметры → Конфиденциальность и безопасность → Для разработчиков → **Режим разработчика = Вкл**
2. Или **запустите сборку от администратора**:  
   ПКМ по `Собрать программу.bat` → **Запуск от имени администратора**

## Способ 2: Сборка через командную строку

1. Нажмите **Win + R**, введите **cmd** и нажмите Enter.
2. Перейдите в папку electron (подставьте свой путь к проекту):
   ```
   cd "D:\Car Service Nikol\electron"
   ```
3. Установите зависимости (один раз):
   ```
   npm install
   ```
4. Соберите программу:
   ```
   npm run build
   ```
5. Готовые файлы будут в папке **dist** внутри **electron**.

---

## Что делать с готовым .exe

- **Nikol CRM Setup 1.0.0.exe** — запустите на том компьютере, где нужна программа, пройдите шаги установки. После этого «Nikol CRM» появится в меню «Пуск».
- **Nikol CRM 1.0.0.exe** — можно просто скопировать на флешку или на другой компьютер и запускать без установки.

Оба варианта требуют подключения к интернету при первом запуске (загружается CRM с сайта).
