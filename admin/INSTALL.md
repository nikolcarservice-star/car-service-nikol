# Установка CRM на компьютер

## Вариант 1: Как программу (PWA) из браузера

1. Откройте CRM в **Chrome** или **Edge**:  
   https://car-service-nikol-crm.vercel.app

2. В меню браузера (три точки) выберите:
   - **Chrome:** «Установить Car Service Nikol — CRM» / «Install Car Service Nikol — CRM»
   - **Edge:** «Приложения» → «Установить этот сайт как приложение»

3. Подтвердите установку. На рабочем столе (или в меню «Пуск») появится ярлык **Nikol CRM**. Приложение открывается в отдельном окне без панели браузера.

## Вариант 2: Отдельный .exe (Electron)

Соберите установщик Windows из папки **electron** в корне проекта — см. **electron/README.md**.

В результате получите файл **Nikol CRM Setup 1.0.0.exe**, который можно установить на любой компьютер с Windows.
